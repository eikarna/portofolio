Tentang Saya

Saya adalah seorang pengembang web & perangkat lunak yang berfokus pada pembuatan situs web/aplikasi responsif dan fungsional. Dengan memiliki pengalaman pengembangan sebuah website atau aplikasi lebih dari 3 tahun, saya berkomitmen untuk memberikan solusi desain yang inovatif dan efisien.

Proyek Saya

GotoPS (GO-lang TO support Private Server)

Sebuah kerangka kerja/framework untuk membuat sebuah server pribadi untuk Game bernama Growtopia. Proyek ini berbasis codecat/go-enet yang telah ditambahkan banyak fitur untuk mendukung berbagai kebutuhan yang menggunakan jaringan ENet.

GotPS

Sebuah proyek server pribadi untuk game bernama Growtopia, menggunakan GotoPS yang membuat server ini siap untuk di kembangkan lebih lanjut dan yang pastinya sudah Production-Ready (thanks to Go for type-safe).

GoDat

Sebuah proyek aplikasi berbasis CLI/terminal untuk serialisasi/deserialisasi sebuah items.dat yang berisi data assets tentang suatu item. Proyek ini hanya untuk game bernama Growtopia.

Smooth Video

Sebuah proyek aplikasi Android untuk memproses suatu video supaya terlihat lebih halus, dengan teknik Motion Interpolation dan Frame Blending menggunakan FFmpegKit sebagai alat pemrosesannya.

Proxy Checker

Sebuah proyek website yang berfungsi untuk cek proxy (HTTP, HTTPS, SOCKS4/SOCKS5) menggunakan Axios dengan TailwindCSS sebagai stylesheeet nya. Proyek ini berbasis Next.js dengan bahasa TypeScript yang membuat proyek ini Production-Ready (Thanks to TypeScript).

Bluetooth Jammer

Sebuah proyek aplikasi Android berbasis Kotlin untuk menyerang perangkat bluetooth terdekat. Proyek ini sedang dalam pengerjaan/WIP.
